# Security Policy

## Supported Versions

These are the current options that support the security policy.

| Option  | Supported                             |
| ------- | ------------------------------------- |
| Games   | :white_check_mark:                    |
| Chats   | :x: (Unless important ones)           |
| Issues  | :white_check_mark:                    |
| Other   | :x: (Unless important)                |
| Reports | :white_check_mark: (On players, etc.) |

## Reporting a Vulnerability

If you find a vulnerability, tell me privately, or if it is non-crititcal, make a Issue on it. Add the critical label onto your issue if you are making a Issue and need it, but don't want to/need to message me.
(@mention me if you need to.)
